Installing Julia & Jupyter

  • Follow the instructions at Install Julia to install juliaup

(for me, all I had to do was enter winget install --name Julia --id 9NJNWW8PVKMN -e -s msstore in the command line but this will be different for different operating systems)

  • Enter julia in the command line to install and run the most recent version of Julia - after a short wait, you will see this

  • Use ] (closing square bracket) to enter package management mode,

  • Enter add IJulia (this package will allow you to use Julia in notebooks):

  • Do the same with Plots, PrettyTables and LaTeXStrings, 
  • Press Backspace,

  • Enter using IJulia and then notebook() to download Jupyter notebooks (confirm by entering y)
  • After installation, Jupyter will launch in the browser

  • You should then be able to create a new folder for your assignments and add A1 Jupyter Notebooks, LaTeX & Julia.ipynb (when it becomes available)

  • To open Jupyter next time, just enter using IJulia and then notebook() again:
